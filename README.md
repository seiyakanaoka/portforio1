# DivIn

## サイト概要
このサイトは、ダイビングの魅力や面白さを発信できるサイトです. 
会員登録してるユーザーは、記事の投稿や閲覧、ユーザー同士のダイレクトメッセージを行えます. 
また、ユーザー同士のダイブプロフィールを閲覧し、どんなダイバーなのか知ることができます. 
会員登録していない方は、投稿の閲覧のみ可能です.
